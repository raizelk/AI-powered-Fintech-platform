## Tech Stack

### **Frontend**

* React.js
* TypeScript
* TailwindCSS
* Recharts (for charts)
* Axios

### **Backend**

* Node.js
* Express.js
* TypeScript
* MongoDB + Mongoose
* JSON Web Tokens (JWT)
* Stripe API (subscriptions)
* Resend API (email service)

### **AI**

* OpenAI API or Gemini API (used for financial insights & categorization)

---

## Features

* User authentication (Sign up / Login / Forgot Password)
* AI-powered financial suggestions
* Create, edit, and delete transactions
* Subscription-based premium features
* Email verification & notifications
* Responsive UI
* Secure API with environment-based configuration
